# Random.first

  • Random.first()

随机生成一个常见的英文名。

Random.first()
// => "Nancy"

# Random.last

  • Random.last()

随机生成一个常见的英文姓。

Random.last()
// => "Martinez"

# Random.name

  • Random.name()
  • Random.name( middle )

随机生成一个常见的英文姓名。

# middle 可选

布尔值。指示是否生成中间名。

Random.name()
// => "Larry Wilson"
Random.name(true)
// => "Helen Carol Martinez"
